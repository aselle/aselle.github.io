Doing your own Windows builds

  1. Clone the git repo
  2. Open a visual studio command prompt
  3. cd into the source directory and make a build directory and cd into it
    1. cd c:\builds\seexpr
    2. md build
    3. cd build
  4. Run CMake c:\program files\cmake 2.8\bin\cmake-gui ..\
  5. Choose a NMake makefile
  6. Click configure and then generate
  7. make install
